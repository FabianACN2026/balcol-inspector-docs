Getting Started

How to Access the Inspector

The BALCOL Inspector lives in the Utility Bar at the bottom of every Lightning page. No setup needed — just click and go.

1
Look at the Utility Bar
At the bottom of your screen you'll see a thin bar with icons. It's always visible on any page.
2
Click "BALCOL Inspector"
The Inspector panel opens from the bottom. Use the tab bar to switch between tools.
3
Resize or pop out
Drag the top edge to resize, or click the expand icon to open in a separate window.
No installation needed. If you can't see it, ask your admin to assign the ABN_Dev_Tools_Admin permission set.

Record Inspector

Inspect any record instantly

Paste a Salesforce Record ID to see every field, its API name, data type, and current value. Edit fields inline without opening the record page.

How to use
1. Open the Record Inspector tab
2. Paste a Record ID (e.g. 001Xx000003GbRd)
3. Browse fields, related lists, and history
4. Click any field value to edit it inline
  • Multi-tab support — inspect up to 8 records side by side
  • Filter fields by name, type, or populated values only
  • Child relationship browsing with record counts
  • Field history tracking with timestamp and user info
  • Single, bulk, and cascade record deletion
SOQL Explorer

Query your data with ease

A full-featured SOQL editor with IntelliSense autocomplete that suggests objects, fields, and relationships as you type — up to 4 levels deep.

How to use
1. Open the SOQL Explorer tab
2. Type SELECT and press Ctrl+Space for autocomplete
3. Press Ctrl+Enter to execute the query
4. Click any cell to edit values inline, or export as CSV
  • Multi-tab queries — up to 8 tabs with independent results
  • Cursor-based pagination with no 2,000 row limit
  • Field chips showing available fields for the queried object
  • Query plan analysis and SOSL support
  • Clickable Record IDs open directly in the Record Inspector
Apex Runner

Execute Apex code instantly

Run anonymous Apex directly from the browser. The runner parses debug logs and surfaces only the output you care about — no more scrolling through raw log files.

How to use
1. Open the Apex Runner tab
2. Write your code or select a snippet from the toolbar
3. Click Execute Anonymous (or press Ctrl+Enter)
4. View parsed System.debug() output below
  • Pre-built code snippets for common operations
  • Browse and download full event logs
  • Compilation error highlighting with line numbers
  • Governor limits monitoring per execution
Caution: Apex runs against live data. DML operations modify real records — always double-check before executing.
Data Loader

Import data from CSV

Upload a CSV file or paste data directly from Excel. A step-by-step wizard guides you through object selection, field mapping, and execution with per-row results.

How to use
1. Open the Data Loader tab and select an object
2. Upload a CSV or paste data from clipboard
3. Map CSV columns to Salesforce fields (auto-mapped when names match)
4. Choose operation: Insert, Update, or Upsert
5. Execute and review per-row success/failure results
  • Drag-and-drop CSV upload or paste from Excel/Sheets
  • Automatic and manual field mapping
  • Batch processing in groups of 200 records
  • Per-row error details for failed records
